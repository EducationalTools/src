# Security Policy

## Supported Versions

The latest version is automatically deployed to all hosts with in 10 minutes of an update, so you should be using the latest version

## Reporting a Vulnerability

Report privately on Github [like so](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability#privately-reporting-a-security-vulnerability)
